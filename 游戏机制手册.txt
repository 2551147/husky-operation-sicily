说明:
  - 软攻(softAtk): 对步兵/工兵/侦察目标的攻击力
  - 硬攻(hardAtk): 对装甲目标的攻击力
  - 防御(def): 通用防御值，受地形加成
  - 移动(mov): 每回合基础移动点数（急行军/撤退x2, 移动射击x0.5）
  - 射程(rng): 攻击距离（炮兵=2可隔格射击，其余=1相邻攻击）
  - 装备(equip): 单位生命值，归零=覆灭
  - 组织(org): 士气值，战斗消耗，过低触发溃退检定
  - 补给消耗(sup): 用于计算增援兵力点和补给总值
  - 视野(vision): 战争迷雾中可见范围（六角格数）

特殊属性:
  - 侦察营: noZOC=true（不产生控制区，不被ZOC减速）
  - 炮兵营: rng=2（唯一可远程攻击的兵种）
  - 工兵营: 在城镇/城市地形中额外+2防御（堑壕加成）

⭐ 老兵系统 (v3.5):
  单位累计消灭2个敌军后晋升为 ⭐老兵，获得永久加成:
    - +1 攻击力 (atkPower += 1)
    - +1 防御力 (defPower += 1)
    - +1 移动力 (maxMoves += 1)
  晋升条件: 累计击杀数 >= 2（kills字段跟踪，每消灭一个敌军 kills+1）
  降级条件: 老兵单位一旦溃退 -> 丧失老兵资格（kills清零，加成全部移除）
  进度显示: 未达2击杀时全UI显示击杀进度
    - 地图兵牌: 0杀=II, 1杀=⚔1(橙), 2杀=★(金)
    - 侧边栏/浮动面板/工具提示均显示 ⚔N/2 或 ⭐老兵
    - 战斗日志和回合战报中单位名前带标记
  意军单位也可晋升老兵（加成与德军/英军相同）

增援费用 (REINF_COST):
  步兵=2兵力点 | 装甲=5 | 炮兵=4 | 工兵=3 | 侦察=3
  意大利步兵=1兵力点（仅德军可部署，Orgx0.60 + 士气-1）

三、六角格坐标系
═══════════════════════════════════════════════════════════════

- 尺寸: COLS=50, ROWS=40
- 类型: Pointy-top (尖顶朝上)
- 坐标: Axial (q, r)
- 游戏六角格大小: HEX_SIZE=28px
- 编辑器六角格大小: HEX_SIZE=18px
- 相邻方向: [+1,0] [+1,-1] [0,-1] [-1,0] [-1,+1] [0,+1]
- 距离公式: (|q1-q2| + |q1+r1-q2-r2| + |r1-r2|) / 2
- Hex Key: `${q},${r}`（用于 Set 存储）

五、战斗系统（v3 — 2d6 + 6级战果表 + 夹击包围）
═══════════════════════════════════════════════════════════════

攻击力计算:
  atkPower = (目标装甲? hardAtk : softAtk)
  atkPower *= Org惩罚 (Org<50%: x0.7, Org<25%: x0.5)
  atkPower *= 补给惩罚 (断补: x0.6)
  atkPower += 组合武器加成 (2格内有己方炮兵且Org>30: +2, 自身非炮兵)
    ⚠ 暴雨天气禁用炮兵支援 (noArtySupport)
  atkPower += 战术卡加成 (突袭命令+3)
  atkPower *= 移动射击惩罚 (isAdvanceFire: x0.5)
  atkPower += 老兵加成 (+1)
  atkDiceBonus = 琴图里佩效果 + 战术卡效果(空中支援+2/炮火支援+3)
  atkDiceBonus -= 守方烟雾掩护减值
  atkDiceBonus -= 天气攻骰惩罚 (暴雨: -1)

夹击/包围 (Flanking):
  攻击时检查守方6个邻格:
    - 每有一个攻方友军(排除攻方自身+溃退+伤亡) -> flankCount++
    - flankCount=1 -> 夹击, atkDiceBonus +1
    - flankCount>=2 -> 包围, atkDiceBonus +2
    - 战斗日志+回合战报标签: 🔗夹击+1骰 / 🔗包围+2骰
    - 炮兵远程射击(距离>=2)也可受益于友军夹击

防御力计算:
  defPower = def + max(0, terrainDef) x 2
  defPower += 森林 +1
  defPower += 工兵堑壕 (城镇/城市 +2)
  defPower += 溃退惩罚 -3
  defPower += 特效加成 (特罗伊纳+2, 拉古萨+1)
  defPower += 战术卡加成 (防御工事+3)
  defPower += 跨河防御加成 (相邻格有河流边: +2)
  defPower += 老兵加成 (+1)
  defPower += 炮兵防御支援 (守方2格内己方炮兵org>30: +2, 炮兵-5org)
    暴雨禁用
  defPower *= Org惩罚 (Org<30: x0.7)
  defPower *= 补给惩罚 (断补: x0.6)

掷骰: 2d6（2-12，钟形曲线，均值7）
  atkTotal = atkPower + roll2d6() + atkDiceBonus
  defTotal = defPower + roll2d6()
  diff = atkTotal - defTotal

6级战果表:
┌──────────┬──────┬──────────┬──────────┬────────────┐
│ 等级     │ diff │ 守方装备 │ 守方组织 │ 攻方损失   │
├──────────┼──────┼──────────┼──────────┼────────────┤
│💥毁灭打击│ >=6  │ max(2,diff/2)│max(15,diffx1.8)│轻微   │
│💢重创    │ >=3  │ max(1,diff/3)│max(10,diffx1.5)│轻微   │
│⚡命中    │ >=0  │ 1        │ max(5,diffx1.2+5)│中    │
│↔️僵持    │ >=-3 │ 1(双方)  │ 各max(3,|diff|x0.5+3)│双方 │
│🔙击退    │ >=-6 │ 0        │ max(2,|diff|x0.3)│中等    │
│💀反攻    │ <-6  │ 0        │ max(1,|diff|x0.2)│严重    │
└──────────┴──────┴──────────┴──────────┴────────────┘

守方特效修正（损伤计算后应用）:
  - 特罗伊纳: 组织损伤 -50% (defOrgSavePct=0.50)
  - 卡塔尼亚(德军守): 组织损伤 /1.20 (defOrgBonus=0.20 -> defMult=1.20)

单位覆灭: equip<=0 或 org<=0 -> removeUnit
  若攻方胜利且为 overrun -> 攻方推进到守方格
  若守方格有战略点 -> 据点易手 + triggerCaptureEffect
炮兵远程无反击: 炮兵在 >=2格距离攻击时，攻方损伤归零
移动射击伤害减半: isAdvanceFire 时 atkPower x 0.5

六、士气与溃退系统
═══════════════════════════════════════════════════════════════

士气检定（战后触发）:
  条件: 战后 Org < 最大Org的30%
  检定: roll2d6() <= max(1, floor(当前Org / 5))
  意军惩罚: 阈值额外-1（nation==='ita' -> threshold = max(1, threshold-1)）

溃退执行 (applyRout):
  1. 标记 routed = true
  2. 撤退1-2格（远离攻击者方向，避开海/山/友军占据格）
  3. Org额外 -15
  4. 若 Org<=0 -> 溃散覆灭

溃退惩罚:
  - ❌ 不能下达攻击或急行军命令
  - 🛡 防御力 -3
  - 💊 每回合仅恢复 5 Org（正常15）
  - 👁 单位卡片红色闪烁边框
  - ⭐ 老兵单位溃退 -> 丧失老兵资格（kills清零，加成全部移除）

溃退恢复 (resolveTurn Phase 5):
  - Org >= 最大Org的50% -> 自动清除溃退标记
  - 战地医院卡: 立即清除所有己方溃退

Org恢复:
  - 溃退单位: +5 Org/回合
  - 正常单位(未移动未战斗): +15 Org(有补给) / +5 Org(无补给)
  - 移动/战斗过的单位: 不恢复
  - ⛈ 暴雨天气: 组织度完全不恢复

七、补给系统（含补给覆盖图/领土覆盖图）
═══════════════════════════════════════════════════════════════

补给源: 己方控制的战略据点 (capture points)

BFS扩散 (calculateSupply):
  - 从每个己方据点出发，BFS在己方领土上扩散
  - supplyUK / supplyDE 双字段分别存储双方补给值（敌我分离）
  - BFS允许通过中立领土(PLAYER.NONE)和有己方部队的格子
  - consolidateTerritory() 在 calculateSupply() 之前执行
  - 基础最大距离: 14 hex
  - 巴勒莫效果: +3 (supplyRadiusBonus)
  - 护航船队卡: +3 (effType: supplyRadius)
  - 道路格: 扩散消耗 0.5
  - ZOC阻断: 邻接敌军单位阻断补给线，侦察兵noZOC不阻断
  - supplyLevel = max(0.25, 1.0 - dist/maxRange)

补给判定:
  - supplyLevel >= 0.25 -> 单位判定为"有补给"
  - 空降补给卡: 指定单位2回合强制有补给（supplyImmune，无视BFS）

断补惩罚（applyAttrition）:
  - 装备 -1 (被包围: 六邻格无友军 -> -2)
  - 组织 -15 (被包围: -30)
  - 装备=0 -> 覆灭

总补给值 (getSupplyValue):
  - sum(己方控制据点.supply) + sum(特殊效果.supplyBonus)
  - 用于计算每回合增援: reinfPoints = max(1, floor(总补给/3))

🔍 补给覆盖图 (Supply Overlay):
  - 底部按钮 "🔍 补给" 或按 S 键切换开关
  - supplyUK/supplyDE 双字段，仅显示当前玩家补给网络，移除FOW限制
  - 地图半透明覆盖:
    · 🟢 绿色 (>=75%): 补给充足
    · 🟡 黄色 (>=50%): 补给中等
    · 🟠 橙色 (>=25%): 补给薄弱
    · ⬛ 暗灰 (<25%): 低/无补给（显示0%）
    · 🔴 红虚线边框: ZOC阻断点
    · ⭐ 发光圈: 补给源（蓝光=英军/红光=德军）
  - 每格标注补给百分比数字
  - 与领土覆盖图互斥（开一个自动关另一个）

🗺 领土归属覆盖图 (Territory Overlay):
  - 底部按钮 "🗺 领土" 或按 T 键切换开关
  - 蓝底+UK标注 = 英军领土，红底+DE标注 = 德军领土，灰底 = 中立
  - 遵守战争迷雾: 可见格全色，已探索半透明，未探索不显示
  - 与补给覆盖图互斥
  - 领土整合(consolidateTerritory): 有己方部队且无敌军邻接的格子自动转为己方领土

八、战争迷雾 (Fog of War)
═══════════════════════════════════════════════════════════════

两层系统: exploredHexes (已探索) + visibleHexes (当前可见)

视野计算 (calculateVisibility):
  - 己方控制据点及邻格: 始终可见
  - 单位BFS视野:
    · 步兵/炮兵/工兵: vision=3
    · 装甲: vision=4
    · 侦察: vision=6
    · 恩纳据点(+3): 位于恩纳的单位视野+3格
    · 山地阻挡: 每格消耗2视野距离

渲染:
  - 当前可见: 正常渲染（地形+敌军+据点标记）
  - 已探索但视野外: 地形灰色显示，敌军隐藏
  - 未探索: 深暗色遮盖
  - 回合执行阶段(RESOLVE): 暂时全图可见
  - 己方单位: 始终可见

工具提示: 视野外的敌军单位显示"❓ 未知单位 (视野外)"
侦察报告卡 (revealAll): 一次性全图开视野

九、战术卡系统 (15张)
═══════════════════════════════════════════════════════════════

开局互选 (Initial Draft):
  1. 英军抽2张 -> 选1张留，另1张给德军
  2. 德军抽2张 -> 选1张留，另1张给英军
  3. 双方各得2张库存

十一、胜利条件
═══════════════════════════════════════════════════════════════

1. 墨西拿沦陷 (Sudden Victory):
   - 英军占领墨西拿 -> 立即胜利

2. 战略胜利 (checkVictory):
   - 英军控制 >= 11个战略点 -> 🇬🇧 英军战略胜利
   - 德军控制 >= 11个战略点 -> 🇩🇪 德军防御胜利
   - 仅当 currentTurn > 5 时触发

3. 歼灭胜利 (Sudden Death):
   - 英军控制点=0 且回合>3 -> 德军压倒性胜利
   - 德军控制点=0 且回合>3 -> 英军压倒性胜利

4. 回合耗尽 (MAX_TURNS=20):
   - 20回合后比占领点数量，多者胜，相等平局
